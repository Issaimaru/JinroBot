# ワンナイト人狼Bot
Discordのチャネルでワンナイト人狼をすることができるようになります

## 使い方
1. [Discord Developer Portal](https://discord.com/developers/applications)にて、Botを作成する
2. [SETTINGS]-[Bot]のページから、トークンをコピーする
3. [Setting.txt]の*Token*に、コピーしたトークンを貼り付ける
4. main.pyを起動する
